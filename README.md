# RaidMine MiniBaritone для Minecraft Fabric 1.21.11

Это не полный оригинальный Baritone. Это простой клиентский автопоиск пути под Fabric 1.21.11.

## Что умеет

- `/mb goto <x> <y> <z>` — построить путь и идти к координатам.
- `/mb stop` — остановить движение.
- `/mb status` — показать состояние.
- `/mb here` — показать текущие координаты игрока.

## Что НЕ умеет

- Не ломает блоки.
- Не ставит блоки.
- Не строит мосты.
- Не копает шахты.
- Не летает.
- Не обходит мобов/лаву как полноценный Baritone.
- Не работает как античит-байпас.

## Как собрать через GitHub без настройки Gradle на ПК

1. Создай новый репозиторий на GitHub.
2. Загрузи туда все файлы из этой папки.
3. Открой вкладку **Actions**.
4. Выбери **Build MiniBaritone 1.21.11**.
5. Нажми **Run workflow**.
6. После сборки скачай artifact **RaidMineMiniBaritone-1.21.11**.
7. Внутри будет `.jar`, его положи в `.minecraft/mods`.

## Как собрать на ПК

Нужно:

- Java 21
- Gradle 8.14.3 или новее

Команда:

```bash
gradle build
```

Готовый `.jar` будет в:

```text
build/libs/
```

## Установка

1. Установи Fabric Loader для Minecraft 1.21.11.
2. Установи Fabric API для 1.21.11.
3. Закинь собранный jar в `.minecraft/mods`.
4. Запусти клиент.

## Настройка версии

Главные версии лежат в `gradle.properties`:

```properties
minecraft_version=1.21.11
yarn_mappings=1.21.11+build.4
loader_version=0.18.1
fabric_version=0.141.2+1.21.11
```
